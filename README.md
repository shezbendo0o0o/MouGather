## Installation

```bash
git clone https://github.com/shezbendo0o0o/MouGather.git
cd MouGather
sudo apt-get update
sudo chmod +x install.sh
sudo bash install.sh
```

---

## Run

```bash
mougather
```

Or:

```bash
python3 mougather.py
```
